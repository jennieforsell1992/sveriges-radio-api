# Kom-igång

1. ladda ner repot från Github
2. Se till att du är i rätt fil i terminalen och skriv sedan in: 
```bash
 npm i 
```
3. för att kunna se projektet i browsern, skriv in i terminalen:
```bash
  npx parcel index.html
```
